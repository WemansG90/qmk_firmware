#Nim's Custom ANSI layout
========================
 ### layer 0 - default layer
  ```
  ,--------------------------------------------------------------------------.
  | Esc|  1 |  2 |  3 |  4 |  5 |  6 |  7 |  8 |  9 |  0 |  - |  = |    \    |
  |--------------------------------------------------------------------------|
  | Tab   |  Q |  W |  E |  R |  T |  Y |  U |  I |  O |  P |  [ |  ] | Bksp |
  |--------------------------------------------------------------------------|
  | Ctrl   |  A |  S |  D |  F |  G |  H |  J |  K |  L |  ; |  ' |  Enter   |
  |--------------------------------------------------------------------------|
  | Shift    |  Z |  X |  C |  V |  B |  N |  M |  , |  . |  / |    RShift   |
  |--------------------------------------------------------------------------|
  |  Fn2  |Fn1| LAlt  |            Space            |RAlt| RGui |App  | Fn3  |
  `--------------------------------------------------------------------------'
  ```
 ### layer 1 - Q/W/ are backlight controls BL_ON / BL_STEP

  ```
   ,--------------------------------------------------------------------------.
  |grave| F1 | F2 | F3 | F4 | F5 | F6 | F7 | F8 | F9 | F10| F11| F12|     Ins|
  |--------------------------------------------------------------------------|
  | CAPS |BlOn|BlUp|    |    |    | Pgup | Home |PgDn|prtsc|sclk|pause| Del  |
  |--------------------------------------------------------------------------|
  |       |Vol-|Vol+|Mute|    |    |left |up |down|right|   |    |           |
  |--------------------------------------------------------------------------|
  |         |Prev|Play|Next|    |    |End  |  /  |  *  | - | + |             |
  |--------------------------------------------------------------------------|
  |       |    |      |                            |    | Stop |    |        |
  `--------------------------------------------------------------------------'
  ```
